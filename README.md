project- Simple Banking Application

A console-based banking application built using Java and Object-Oriented Programming concepts.

 Features-
- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- Handles invalid input and insufficient balance

 Technologies Used-
- Java
- IntelliJ IDEA
- Git & GitHub

 How to Run-
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run `SimpleBankingApplication.java`
